# Onchain - web2-inspo -> web3-impl

Implementação inicial de:

- Contrato Solidity `contracts/GamlrOnchainRegistry.sol`
- JS SDK `sdk/src/index.js`

A proposta converte os objetos de save/profile inspirados em Play Games para referências on-chain via URI (IPFS/Arweave/HTTPS), evitando armazenar binário bruto na chain.
